// FEATURED BUILD · OUR FIRST APP

CASIDOM

Family management, gamified.

The first product we built end‑to‑end — a family hub where parents set tasks and attach rewards, and kids earn points they spend on real things.

iOS + ANDROID DESIGN + CODE BUILT FROM ZERO
THE PROBLEM

Chores, allowance and family plans lived across sticky notes, group chats and nagging. Kids had no reason to follow through, and parents had no single place to run it.

THE SOLUTION

A single family hub with a token economy. Parents assign tasks and attach a reward; kids complete them, earn tokens, and swap them in a shop for real things — screen time, treats, money, a trip. Motivation, built in.

// WHAT WE BUILT

One complete family hub.

Parent & child roles

Parents create and confirm tasks and manage rewards. Kids complete tasks, earn tokens and shop.

Token reward shop

Each family sets its own currency and catalog — an hour of gaming, ice cream, 1000₽, a weekend trip.

Family leaderboard

A little friendly competition. Everyone sees who’s earning, ranked by tokens.

Shared calendar

Birthdays, trips and plans in one place, so the whole family stays in sync.

THE RESULT

Casidom shipped as a complete cross‑platform product — designed and built from zero. Onboarding, roles, a live token economy, reward shop, leaderboard and a shared calendar, all in one family hub.
